# Reports Directory

Generated PDF reports and visualizations will be saved here.

## Contents

After running analysis, this directory will contain:

- `Vigil_Report_YYYY-MM-DD.pdf` - Main PDF report
- `report_metadata.json` - Report metadata
- `chart_*.png` - Generated visualization charts

## Structure

```
reports/
├── Vigil_Report_2025-11-10.pdf
├── report_metadata.json
├── chart_revenue_20251110_143022.png
├── chart_churn_rate_20251110_143025.png
└── anomaly_summary_20251110_143030.png
```
